Тест курсоров

Кнопки (должен быть курсор pointer):

Ссылки (должен быть курсор pointer):

Ссылка 1
Ссылка 2

Поля ввода (должен быть курсор text):

Инструкции:

1. Наведите мышь на кнопки - курсор должен стать "pointer" (рука)

2. Наведите мышь на ссылки - курсор должен стать "pointer" (рука)

3. Наведите мышь на поля ввода - курсор должен стать "text" (I-образный)

4. Курсор должен изменяться на всей области элемента, а не только на границе
